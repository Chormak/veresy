# Veresy

**Veresy** is a cross-platform desktop application for managing repair orders in a computer service center.

The project is currently in development and aims to simplify workflow management, order tracking, inventory control, and client communication.

Built with **C++** and **Qt6**, targeting both **Windows** and **Linux**.

---

## ✨Features (Planned)✨

### 🧾Order Management🧾

* Fast order creation (target: under 30 seconds)
* Real-time order queue
* Drag-and-drop status management
* Repair stages tracking
* Order history

### 🔍Search & Navigation🔍

* Instant search across orders
* Hotkeys for fast workflow
* Split-screen order view

### 👨‍🔧Team Workflow👨‍🔧

* Sidebar with current staff activity
* Task distribution visibility

### 📦Inventory📦

* Integrated warehouse system
* Automatic parts write-off
* Low stock notifications

### 📡Client Communication📡

* Telegram integration (via API)
* Message templates
* Full communication history
* Automatic notifications:

  * Order ready
  * Delays
  * Waiting for parts

### 📷Additional Tools📷

* Photo attachments to orders
* USB barcode scanner support (IMEI)
* Auto-detection of device models

### 💰Finance & Documents💰

* Cost estimation and invoice generation (one click)
* Receipt printing
* Online payment support (planned)

### 📊Analytics📊

* Dashboard with statistics
* One-click reports

### 🔐Access Control🔐

* Role-based permissions system

---

## 🛠Tech Stack🛠

* **Language:** C++
* **Framework:** Qt6
* **Database:** SQL (Qt SQL module)
* **Platform:** Windows / Linux
